Хакатон 2026 - Баг-репорты
Проект: Запись на приём (CloudCom)
Даты: 25–26 апреля 2026

О проекте
Тестировали веб-приложение для записи граждан в госучреждения: МФЦ, поликлиники, ГИБДД, ЗАГС, налоговую и другие. В приложении можно искать учреждения, выбирать услугу и время, записываться на приём, смотреть и отменять свои записи, регистрироваться, редактировать профиль и отправлять обратную связь.

Что делали
Проверяли сам сайт и API через Swagger. Проходили основные сценарии, вводили некорректные данные, искали дыры в безопасности, смотрели как приложение ведёт себя в нестандартных ситуациях. Большинство багов нашли через прямое взаимодействие с API, часть - через интерфейс и инструменты разработчика в браузере.

Результаты
Всего нашли 11 багов. Четыре серьёзных, остальные - незначительные и визуальные. Среди находок: аккаунт создаётся без пароля, можно отменить чужую запись, можно смотреть чужие записи через подмену параметра в адресной строке, проверка паролей сравнивает только длину строк, имя в профиле сбрасывается после обновления страницы.

Структура
В папке bugs лежат файлы с полным описанием каждого бага. Названия файлов отражают суть проблемы.
