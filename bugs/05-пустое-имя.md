Можно сохранить пустое отображаемое имя

Серьёзность: Незначительный
Страница: Профиль

Шаги:
1. Открыть профиль
2. Нажать "Редактировать"
3. Удалить всё из поля "Имя"
4. Нажать "Сохранить"

Ожидаемый результат:
Ошибка "Имя не может быть пустым".

Фактический результат:
Пустое имя сохраняется без ошибок.
