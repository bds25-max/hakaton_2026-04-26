Можно отменить чужую запись на приём

Серьёзность: Серьёзный
Страница: API /api/appointments/{id}/cancel

Шаги:
1. Создать запись от имени victim
2. Запомнить id записи
3. От имени demo1 выполнить POST /api/appointments/{id}/cancel?user_id=demo1

Ожидаемый результат:
Ошибка 403 "Нельзя отменить чужую запись".

Фактический результат:
200 OK, запись отменена.
