Аккаунт создаётся без пароля

Серьёзность: Серьёзный
Страница: Регистрация

Шаги:
1. Открыть страницу регистрации
2. Заполнить все поля, включая пароль и подтверждение
3. Открыть DevTools (F12), вкладка Network
4. Нажать "Зарегистрироваться"
5. Посмотреть тело запроса во вкладке Payload

Ожидаемый результат:
В теле запроса есть поля password и confirmPassword.

Фактический результат:
В теле запроса только login и display_name. Пароль не передаётся, аккаунт создаётся без пароля.
