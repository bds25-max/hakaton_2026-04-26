Повторная отмена уже отменённой записи возвращает 200

Серьёзность: Незначительный
Страница: API /api/appointments/{id}/cancel

Шаги:
1. Создать и отменить запись
2. Повторно выполнить отмену той же записи

Ожидаемый результат:
Ошибка 400 "Запись уже отменена".

Фактический результат:
200 OK, дата cancelled_at обновляется.
